# yakir-cohen-productions

אתר Next.js עבור **יקיר כהן הפקות**.

## פיתוח מקומי

דרישות: Node.js (מומלץ גרסה התואמת ל־`package.json`).

```bash
npm install
npm run dev
```

השרת המקומי: [http://localhost:3000](http://localhost:3000)

## משתני סביבה

העתק את `.env.example` ל־`.env.local` ומלא את הערכים הרלוונטיים (כתובת האתר, אנליטיקס, StemSplit וכו').

## בנייה לפרודקשן

```bash
npm run build
npm start
```
