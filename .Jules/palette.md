## 2025-05-15 - [Consistent Clipboard Feedback]
**Learning:** Providing immediate visual ("הועתק!") and haptic feedback for clipboard actions significantly improves user confidence in the interaction, especially on mobile. Standardizing this pattern across all share/copy components (ShareButton, SocialShare) creates a predictable UX.
**Action:** Always implement a temporary success state with haptic feedback for any "copy to clipboard" action in this design system.
